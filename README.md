
Este repositorio contiene una página web informativa creada como práctica de la Unidad 5.

El proyecto está dedicado a mostrar los tipos de vermuts, sus características principales y algunos datos curiosos sobre esta bebida tan popular.
El objetivo es aplicar los conocimientos de HTML y CSS en la creación de una web estática y funcional, publicada mediante GitHub Pages.

Contenido del repositorio
	•	index.html → Página principal del sitio.
	•	style.css → Hoja de estilos para el diseño visual.
	•	img/ → Carpeta con imágenes relacionadas con el tema.
	•	README.md → Documento con información general del proyecto.

 Tecnologías utilizadas
	•	HTML5 → para la estructura y el contenido de la web.
	•	CSS3 → para el diseño, colores, tipografía y presentación.
