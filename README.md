This repository contains an informational webpage created as a practice for Unit 5.

The project is dedicated to showcasing the different types of vermouth, their main characteristics, and some interesting facts about this popular beverage.
The main goal is to apply the knowledge of HTML and CSS to build a static and functional website, published through GitHub Pages.

Repository Contents
	•	index.html → Main page of the website.
	•	style.css → Stylesheet for visual design.
	•	img/ → Folder with images related to the topic.
	•	README.md → This file containing general project information.
Technologies Used
	•	HTML5 → for the structure and content of the website.
	•	CSS3 → for design, colors, typography, and layout.

How to View the Website
You can directly access the published version through GitHub Pages at the following link:
https://lidmo11.github.io/Exercise_Unit5_website/￼
Or, if you prefer, you can open it locally:
	1.	Download or clone this repository.
	2.	Open the index.html file with any web browser (e.g., Chrome, Edge, or Safari).
